*  What Is Last Odyssey?: A little bit about this game and its principles.
*  Getting Started: Advice for setting up new games.
*  Advice for Players: How to be a good player while at the table.
*  Advice for GMs: How to run Last Odyssey so it meets its full potential.
*  Advice for Facilitators: Everything that goes into keeping a game running.
